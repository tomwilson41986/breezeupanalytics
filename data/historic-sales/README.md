# Historic Sales Data

This folder contains historic sales data files used for analytics, statistics, and visualization.

## Purpose

- Store historical auction/sale records for long-term analysis
- Power analytics dashboards and reporting tools
- Enable trend analysis and statistical modeling

## Supported Formats

Upload data files in any of these formats:
- **CSV** (`.csv`) — comma-separated values
- **JSON** (`.json`) — structured records
- **Excel** (`.xlsx`) — spreadsheet files

## Usage

Place your historic sales data files directly in this folder. They will be tracked in version control and available for analytics processing.
